# StreeRaksha 🛡️
> A comprehensive women's safety Android application that empowers users with real-time protection, emergency response, and community-aware safety tools.

![Platform](https://img.shields.io/badge/Platform-Android-green) ![Status](https://img.shields.io/badge/Status-Completed-brightgreen) ![Language](https://img.shields.io/badge/Language-Java%2FKotlin-orange)

## Features
- **One-Tap SOS** — Instantly alerts emergency contacts and sends live location via SMS
- **Real-Time Location Sharing** — Powered by Google Maps API
- **Hidden Camera Detection** — Scans surroundings for covert surveillance devices
- **One-Click Helpline Calls** — No manual dialing; directly connects to national emergency numbers
- **Nearby Safety Places** — Locates police stations, hospitals, pharmacies, and shelters
- **Loud Siren** — Triggers an instant audio alarm as a deterrent
- **Emergency Contacts Management** — Add and manage trusted contacts with ease
- **User Profile** — Personalized setup for quick emergency response

## Tech Stack
`Java/Kotlin` · `Android SDK` · `Firebase` · `Google Maps API` · `SMS API` · `GPS/Location Services`

## Getting Started
```bash
git clone https://gitlab.com/pavithrabhat09/streeraksha
# Add Google Maps API key in AndroidManifest.xml
# Add Firebase google-services.json to /app
# Open in Android Studio → Build → Run
```

## Why StreeRaksha?
Women's safety remains a critical challenge in public spaces. StreeRaksha bridges the gap between danger and help — putting powerful safety tools a single tap away.
