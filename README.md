## What is StreeRaksha?

Women's safety shouldn't depend on luck or timing. StreeRaksha is an Android app built to put real power in a woman's hands - whether she's walking home late, traveling alone, or simply feels unsafe. From a one-tap SOS to hidden camera detection, every feature is designed to act fast when it matters most.

---

## Features

-  **One-Tap SOS** - Instantly alerts emergency contacts and sends live location via SMS
-  **Real-Time Location Sharing** - Continuous location tracking powered by Google Maps API
-  **Hidden Camera Detection** - Scans the surroundings for covert surveillance devices
-  **One-Click Helpline Calls** - Directly connects to national emergency numbers, no dialing needed
-  **Nearby Safety Places** - Finds police stations, hospitals, pharmacies, and shelters nearby
-  **Loud Siren** - Triggers an instant audio alarm to deter threats
-  **Emergency Contacts** - Easily add and manage your trusted contacts
-  **User Profile** - Personalized setup for faster emergency response

## Getting Started

```bash
git clone https://github.com/your-username/streeraksha
```

1. Open the project in **Android Studio**
2. Add your **Google Maps API key** in `AndroidManifest.xml`
3. Place the **`google-services.json`** file inside `/app`
4. Hit **Build → Run**
